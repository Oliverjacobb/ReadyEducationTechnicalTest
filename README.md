# ReadyEducationTechnicalTest
This is an Angular v16 application that retrieves a list of users from an API.

### Getting Started

To run the application you will need to have the following installed...

- Node.js
- Angular CLI
- NPM package manager

Once you have those installed, you can run the application by following these steps:

1. Clone the repository
2. Navigate to the user-app directory
3. Run npm i to install dependencies
4. Run ng serve to start the dev server

The application should now be running on http://localhost:4200

### Features

The application has the following features:

- Retrieving list of users from API
- Search for user by name or country
- Find user location on google maps
- See user timezone

### Running Tests
1. Navigate to the user-app directory
2. Run ng-test to start the Karma test runner, which will execute the jasmine tests

Currently the only tests for this application are the out of the box Angular tests.
